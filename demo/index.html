flashMe

Another pointless plugin.

Make me flash all kinds of colors.

Usage

Standard usage with default settings

Code
$('#example1').flashMe();
Preview
Example 1

Apply transition

Code
$('#example2').flashMe({ transition: 500 });
Preview
Example 2

Flash the background too!?

Code
$('#example3').flashMe({ backgroundColors: "surprise" });
Preview
Example 3

Specify the interval

Code
$('#example4').flashMe({ interval: 100 });
Preview
Example 4

Specify the colors to flash with. (same applies to backgroundColors)

Code
$('#example4').flashMe({ colors: ["#43fa4e", "#2afdec", "#4ecdae"] });

Options

Option Default Values
colors "surprise" false (don't flash the color attribute)
array of colors (["#43fa4e", "#2afdec", "#4ecdae"])
backgroundColors "surprise" false (don't flash the background-color attribute)
array of colors (["#43fa4e", "#2afdec", "#4ecdae"])
interval 1000 interval between flashes in milliseconds
transition false transition speed in milliseconds
